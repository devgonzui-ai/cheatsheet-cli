# Vim チートシート

## モード切替

| キー | 説明 |
|------|------|
| `i` | カーソル位置で挿入モード |
| `I` | 行頭で挿入モード |
| `a` | カーソル後ろで挿入モード |
| `A` | 行末で挿入モード |
| `o` | 下に行挿入して挿入モード |
| `O` | 上に行挿入して挿入モード |
| `v` | ビジュアルモード |
| `V` | 行選択ビジュアルモード |
| `Ctrl+v` | 矩形選択ビジュアルモード |
| `Esc` | ノーマルモードに戻る |

## カーソル移動

| キー | 説明 |
|------|------|
| `h/j/k/l` | 左/下/上/右 |
| `w` | 次の単語の先頭 |
| `b` | 前の単語の先頭 |
| `e` | 単語の末尾 |
| `0` | 行頭 |
| `^` | 行頭(空白除く) |
| `$` | 行末 |
| `gg` | ファイル先頭 |
| `G` | ファイル末尾 |
| `:{数字}` | 指定行にジャンプ |
| `%` | 対応する括弧へ |
| `f{文字}` | 行内で文字を前方検索 |
| `F{文字}` | 行内で文字を後方検索 |

## 編集

| キー | 説明 |
|------|------|
| `x` | 1文字削除 |
| `dd` | 行削除 |
| `dw` | 単語削除 |
| `d$` | カーソルから行末まで削除 |
| `D` | d$と同じ |
| `cc` | 行を変更 |
| `cw` | 単語を変更 |
| `C` | カーソルから行末まで変更 |
| `yy` | 行をヤンク(コピー) |
| `yw` | 単語をヤンク |
| `p` | カーソル後にペースト |
| `P` | カーソル前にペースト |
| `u` | アンドゥ |
| `Ctrl+r` | リドゥ |
| `.` | 直前の操作を繰り返し |
| `r{文字}` | 1文字置換 |
| `R` | 置換モード |
| `J` | 行を連結 |
| `~` | 大文字小文字切替 |
| `>>` | インデント |
| `<<` | アンインデント |

## 検索・置換

| キー | 説明 |
|------|------|
| `/{パターン}` | 前方検索 |
| `?{パターン}` | 後方検索 |
| `n` | 次の検索結果 |
| `N` | 前の検索結果 |
| `*` | カーソル下の単語を前方検索 |
| `#` | カーソル下の単語を後方検索 |
| `:s/old/new/` | 行内で最初の一致を置換 |
| `:s/old/new/g` | 行内で全て置換 |
| `:%s/old/new/g` | ファイル全体で置換 |
| `:%s/old/new/gc` | 確認しながら置換 |

## ファイル操作

| コマンド | 説明 |
|----------|------|
| `:w` | 保存 |
| `:w {ファイル名}` | 名前を付けて保存 |
| `:q` | 終了 |
| `:q!` | 保存せず終了 |
| `:wq` / `:x` / `ZZ` | 保存して終了 |
| `:e {ファイル}` | ファイルを開く |
| `:r {ファイル}` | ファイル内容を挿入 |
| `:!{コマンド}` | 外部コマンド実行 |

## ウィンドウ分割

| キー | 説明 |
|------|------|
| `:sp` | 横分割 |
| `:vs` | 縦分割 |
| `Ctrl+w h/j/k/l` | ウィンドウ移動 |
| `Ctrl+w w` | 次のウィンドウ |
| `Ctrl+w q` | ウィンドウを閉じる |
| `Ctrl+w =` | ウィンドウサイズ均等化 |
| `Ctrl+w +/-` | 高さ増減 |
| `Ctrl+w >/<` | 幅増減 |

## タブ

| コマンド | 説明 |
|----------|------|
| `:tabnew` | 新規タブ |
| `:tabn` / `gt` | 次のタブ |
| `:tabp` / `gT` | 前のタブ |
| `:tabc` | タブを閉じる |

## マクロ

| キー | 説明 |
|------|------|
| `q{レジスタ}` | マクロ記録開始 |
| `q` | マクロ記録終了 |
| `@{レジスタ}` | マクロ実行 |
| `@@` | 直前のマクロを再実行 |

## その他

| キー | 説明 |
|------|------|
| `:set nu` | 行番号表示 |
| `:set nonu` | 行番号非表示 |
| `:noh` | 検索ハイライト解除 |
| `Ctrl+g` | ファイル情報表示 |
| `za` | 折りたたみ切替 |
