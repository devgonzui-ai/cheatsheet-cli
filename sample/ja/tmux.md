# tmux チートシート

## セッション管理

| コマンド | 説明 |
|---------|------|
| `tmux new -s 名前` | 名前付きセッション作成 |
| `tmux ls` | セッション一覧 |
| `tmux attach -t 名前` | セッションにアタッチ |
| `tmux kill-session -t 名前` | セッション終了 |
| `tmux rename-session -t 旧 新` | セッション名変更 |

## プレフィックスキー (デフォルト: Ctrl+b)

| キー | 説明 |
|------|------|
| `d` | セッションからデタッチ |
| `$` | セッション名変更 |
| `s` | セッション一覧 |
| `(` | 前のセッション |
| `)` | 次のセッション |

## ウィンドウ操作

| キー | 説明 |
|------|------|
| `c` | 新規ウィンドウ作成 |
| `,` | ウィンドウ名変更 |
| `&` | ウィンドウ削除 |
| `n` | 次のウィンドウ |
| `p` | 前のウィンドウ |
| `0-9` | 番号でウィンドウ切替 |
| `w` | ウィンドウ一覧 |
| `f` | ウィンドウ検索 |

## ペイン操作

| キー | 説明 |
|------|------|
| `%` | 横に分割 |
| `"` | 縦に分割 |
| `o` | 次のペインへ移動 |
| `q` | ペイン番号表示 |
| `x` | ペイン削除 |
| `z` | ペインのズーム切替 |
| `{` | ペインを左に移動 |
| `}` | ペインを右に移動 |
| `Space` | レイアウト切替 |
| 矢印キー | ペイン間移動 |

## ペインのリサイズ (プレフィックス + 長押し)

| キー | 説明 |
|------|------|
| `Ctrl + 矢印` | 1セルずつリサイズ |
| `Alt + 矢印` | 5セルずつリサイズ |

## コピーモード (プレフィックス + [)

| キー | 説明 |
|------|------|
| `Space` | 選択開始 |
| `Enter` | コピー |
| `q` | コピーモード終了 |
| `/` | 前方検索 |
| `?` | 後方検索 |
| `n` | 次の検索結果 |
| `N` | 前の検索結果 |

## その他

| キー | 説明 |
|------|------|
| `?` | キーバインド一覧 |
| `:` | コマンドプロンプト |
| `t` | 時計表示 |
| `i` | ウィンドウ情報表示 |
